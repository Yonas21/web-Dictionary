This is Simple Dictionary Application that is based on model-view-controller(mvc) architectural pattern
build using express and some package like crypto to hash password for security
here are some guidelines to use this app 
1,install node.js from www.nodejs.com
2.install express using node package manager(npm) npm install -g express ,-g for installing the package globally on your machine for future use
3.run the app using node express.js and run on your browser like this
->localhost:3000/the file in the public folder.
